# Logistic---Women_Affairs_Dataset
Predicting if a women is having an extra marital affair based on the set of given features

Features considered -
1. rate_marriage
2. age
3. yrs_married
4. children
5. religious
6. educ
7. occupation
8. occupation_husb

zip()
